Es un lenguaje de programación de alto nivel, de código abierto y orientado a objetos,  diseñado para ser fácil de leer y escribir para los humanos. 

Fue creado por Guido van Rossum y lanzado por primera vez en 1991. El nombre proviene de la serie de comedia británica “Monty Python’s Flying Circus”.

Es un lenguaje de programación de propósito general, es decir que sirve para realizar: desarrollo web, análisis de datos, aprendizaje automático,  automatización, web scraping, etc.

Una librería en Python es un conjunto de módulos con funciones y clases reutilizables para tareas específicas.
