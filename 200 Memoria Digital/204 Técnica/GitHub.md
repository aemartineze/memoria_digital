GitHub es una plataforma web para alojar repositorios [[Git]] de forma remota. Permite guardar el código en la nube, sincronizarlo con repositorios locales y colaborar con otras personas mediante control de versiones.

Funciona como un servidor Git accesible por internet: el repositorio local vive en tu computadora y GitHub almacena una copia remota con historial, cambios y versiones.
## Para qué se usa

- Guardar código de forma remota.
- Sincronizar trabajo entre distintos equipos.
- Compartir y colaborar en proyectos.
- Mantener un historial de cambios accesible.
## Relación con Git

Git es la herramienta de control de versiones que se ejecuta localmente.  
GitHub es un servicio que aloja repositorios Git.

Se puede usar Git sin GitHub, pero no GitHub sin Git.

## URL
https://github.com

## Uso básico
- Crear repositorios remotos.
- Conectar repositorios locales mediante `git remote`.
- Enviar cambios (`git push`) y traer cambios (`git pull`).
